# Calendar App

A simple and integrated calendar for tracking your important events and birthdays.

## Features
*   **Event Tracking**: Add holidays, meetings, and work deadlines.
*   **Birthday Reminders**: Never forget a birthday!
    *   Birthdays are highlighted in **Orange**.
    *   Includes year tracking to calculate age.
*   **Visual Highlights**: Events are color-coded on the calendar view.
    *   Birthdays: Orange
    *   Meetings: Blue
    *   Holidays: Green
*   **List View**: See all your upcoming events in a chronological list.

## Usage
*   Click **+ Add Event** to create a new entry.
*   Select a **Category** to apply color coding.
*   Select an event and click **Delete Selected** to remove it.
