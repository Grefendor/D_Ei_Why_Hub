# Task Board

Stay organized with the Task Board, designed for shared households and teams.

## Features
*   **Recurring Tasks**: Set tasks to repeat **Daily**, **Weekly**, or **Every 3 Days**.
*   **Smart Assignments**:
    *   **Specific People**: Assign to one or more specific users (e.g., "Alice & Bob").
    *   **Any N People**: Require any 2 people to complete the task.
    *   **All People**: Everyone must complete it (e.g., "Team Meeting").
*   **Completion Tracking**:
    *   Tasks disappear when completed and reappear when due again.
    *   "Show Completed" toggle lets you see what's been done today.
*   **People Management**: Add and remove users from the system.

## Usage
1.  **Add People**: Go to "People" to add family members or roommates.
2.  **Add Task**: Define the task name, frequency, and who needs to do it.
3.  **Complete Task**: Click "Done". If the task requires a specific person, it will ask "Who are you?".
